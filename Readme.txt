README TUGAS KECIL 1 STRATEGI ALGORITMA

DESKRIPSI PROGRAM :

Program sederhana untuk menemukan solusi dari persamaan cryptarithmetic, ditulis dalam bahasa pemrograman Python

REQUIREMENT PROGRAM :

Memiliki Python versi 3.7 keatas

CARA MENGGUNAKAN PROGRAM :

1. Masuk ke dalam folder bin, kemudian launch cryptarithmetic_permutation.exe (pastikan Windows Defender Anda sudah off)
2. Apabila executable file masih tidak bisa dijalankan, bukalah cmd lalu masuk ke dalam folder src, kemudian jalankan "py cryptarithmetic_permutation.py" untuk menjalankan Program
3. Apabila program sudah berjalan, masukkan nama file sesuai dengan nama file yang ada dalam folder test
4. Program akan menjalankan komputasi untuk mencari solusi dari persamaan cryptarithmetic_permutation
5. Setelah program selesai mencari solusi, Anda akan diberi pertanyaan untuk memberi input file lain atau tidak, jika masih ingin, silahkan masukkan 'Y', jika tidak masukkan 'N'
